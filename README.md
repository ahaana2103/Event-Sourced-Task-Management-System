# Event-Sourced-Task-Management-System
Event-Sourced Task Management System built with Node.js and Express.js that records task updates as events and reconstructs state from the event log.

Event Sourcing Task Manager

📌 Project Description

This project is a simple Task Manager built using Event Sourcing architecture.
Instead of storing the current state of tasks, all changes are stored as events.

---

🚀 Features

- Create Task
- Update Task
- Complete Task
- Delete Task
- Rebuild State from Events

---

🧠 What is Event Sourcing?

In this approach:

- We store events (history) instead of final data
- Events include:
  - TaskCreated
  - TaskUpdated
  - TaskCompleted
  - TaskDeleted

👉 The current state is generated by replaying all events

---

🔁 What is Rebuild?

Rebuild means:

«Recreating the current state by replaying all stored events»
